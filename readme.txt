Verkoop je eigen boek of ontwerp met print-on-demand! Plaats eenvoudig producten
uit je Print API account op je site met deze plugin.

== Description ==

Heb je een boek geschreven en wil je het online verkopen? Wil je jouw
fotografie online aanbieden op canvas? Heb je misschien tegeltjes ontworpen om
te verkopen? Print API maakt het makkelijk.

1. Maak een account op [printapi.nl](https://www.printapi.nl/mogelijkheden/web-plugin "Print API").
1. Upload je foto of PDF en kies een prijs.
1. Je ontvangt een bestel-link voor je creatie.

Met deze plugin plaats je daarna eenvoudig een bestelknop op je website, met
automatische thumbnail en prijsindicatie.

**Instructie**

Plugin geïnstalleerd? Je kunt nu bestelknoppen invoegen met de nieuwe *Print
API* knop in je pagina editor. Deze genereert automatisch de benodigde `[printapi]`
shortcode.

Voor geavanceerd gebruik accepteert deze shortcode de volgende attributen:

- `class` – Extra CSS classes voor het element.
- `title` – Aangepaste titel-tekst voor de bestelknop.

== Installation ==

1. Upload de plugin-bestanden naar de `/wp-content/plugins/printapi` map.
1. Activeer de plugin via het 'Plugins' scherm in WordPress.
1. De `[printapi]` shortcode en knop zijn nu actief.

== Frequently Asked Questions ==

= Wat zijn de kosten / wat levert het op? =

Het gebruik van Print API is helemaal gratis. Je bepaalt zelf een winstmarge op
de inkoopprijs van ons drukwerk. Die winst wordt maandelijks uitbetaald.

= Is het uiterlijk van de bestelknop aanpasbaar? =

Ja, vooralsnog via CSS. De bestelknop is uitgebreid voorzien van CSS-classes
zodat je deze helemaal naar eigen wens kunt aanpassen. Tevens is de source code
natuurlijk vrij aanpasbaar.

= Is het uiterlijk van het bestelproces aanpasbaar? =

Ja, via je account kun je zelf een hoofdkleur, achtergrond en logo kiezen.

= Welke betaalmethodes ondersteunt het bestelproces? =

Print API ondersteunt momenteel betalingen via iDeal en CreditCard. Extra
betaalmethodes worden dit jaar verwacht.

= Kan mijn bezoeker ook zelf het ontwerp aanleveren? =

Niet met de plugin, maar bekijk zeker eens onze [REST API](https://www.printapi.nl/mogelijkheden/rest-api "Print API - REST API")
en/of [Online Editor](https://www.printapi.nl/mogelijkheden/online-editor "Print API - Online Editor").
